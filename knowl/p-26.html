Some expressions may not be linear, using exponents other than one on numerous variables. Note that using multiple variables does not necessarily make the expression non-linear, but involving elements like \(x^2\text{,}\) \(\sqrt{\alpha}\text{,}\) and \(\dfrac{1}{\heartsuit}\) are certainly non-linear when \(x\text{,}\) \(\alpha\text{,}\) \(\heartsuit\) are variable. This does not change how we address using a reasonable order of operations for expanding or condensing a complicated expression. The only alteration might be additional considerations for what it means to combine "like terms".
